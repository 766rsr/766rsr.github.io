遊戲名稱：SURVIVORS
遊戲類型：類生存者遊戲
遊戲操作：利用上下左右鍵進行移動，武器會自動進行攻擊，擊破敵人後可拾取掉落經驗，並藉此升級武器與被動能力。

武器種類：
1. 領域：於玩家四周隨機地點生成，生成點附近造成傷害並存留一段時間。
2. 立場：於玩家四周張開防護立場來造成傷害，並跟隨玩家移動。
3. 追蹤：會朝四周丟出追蹤子彈，子彈會自動追蹤離自己最近的敵人。
4. 守衛：生成兩個守衛並以玩家為中心，持續繞行旋轉。
5. 雷射：會於上下左右持續射出雷射，於短時間造成大傷害。

被動種類
1. Might：角色基礎傷害
2. Armor：角色傷害減免
3. Recovery：角色自然恢復量
4. Cooldown：技能冷卻速度
5. MoveSpeed：角色移動速度
6. Growth：經驗成長速度
